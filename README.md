# Mall 3D Simulation - OpenGL Project

مشروع رسومي ثلاثي الأبعاد لمحاكاة مول تجاري باستخدام **OpenGL** و **C++**، يحتوي على أربع مناطق رئيسية: صالة ألعاب، مكتبة، متجر ملابس، ومتجر أدوات منزلية.

---

## 🧩 المكونات الرئيسية

### 🎮 صالة الألعاب
- بيئة تفاعلية لمحاكاة صالة ألعاب
- عرض الأجهزة والأنشطة الترفيهية
!)

### 📚 المكتبة
- أرفف كتب منظمة
- بيئة هادئة للتصفح
!)

### 👗 متجر الملابس
- عرض ملابس باستخدام مجسمات
- تصميم واقعي للمتاجر الحديثة
!)

### 🏠 متجر الأدوات المنزلية
- عرض الأدوات المنزلية والأثاث
- تنظيم العناصر بشكل واضح
!)

---

## ⚙️ التقنيات المستخدمة
- OpenGL للرسم ثلاثي الأبعاد
- لغة C++  
- إضاءة (Lighting)، خامات (Textures)، التحريك (Transformations)، الكاميرا (Camera Navigation)

---

## ▶️ طريقة التشغيل
1. فتح المشروع باستخدام **Visual Studio**
2. التأكد من تثبيت مكتبات OpenGL
3. تشغيل المشروع عبر **Run**
